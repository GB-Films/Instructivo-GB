# Instructivo-GB
Una pagina web que resuma el instructivo para los diferentes requerimientos de la empresa en los proyectos de producción de contenidos
# Info administrativa · Gran Berta S.R.L.

Pequeño sitio estático con la información administrativa básica de la productora
(razón social, datos para contratos, criterios de pagos, garantías, carga de facturas y contacto).

Pensado para uso interno, hosteado en GitHub Pages.

## Estructura

- `index.html`  
- `assets/css/styles.css`  
- `assets/js/main.js`

## Cómo publicar en GitHub Pages

1. Crear el repositorio (por ejemplo `gb-admin-info`).
2. Subir todos los archivos manteniendo la misma estructura de carpetas.
3. En GitHub, ir a **Settings → Pages**.
4. En "Source", elegir la rama (por ejemplo `main`) y la carpeta `/root`.
5. Guardar. GitHub va a generar una URL del estilo:

   `https://TU-USUARIO.github.io/gb-admin-info/`

Compartís ese link y listo.
